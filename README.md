# Beego sns auth

Social accounts connect for beego

### Support Platform

* Github
* Google+
* Sina Weibo
* Tencent QQ
* Dropbox
* Facebook
* Weixin
* Renren

### Installation

```
go get github.com/philsong/sns-auth
```

### Features

* All use Beego
* Support multiple sns platform.
* Multiple sns accounts can be associated to a single user.
